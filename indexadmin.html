Los cambios realizados aquí se guardan localmente en memoria. Para publicarlos en el sitio web permanentemente, debes subir los archivos actualizados a tu repositorio de GitHub y hacer deploy en Netlify. Si usas Decap CMS con Git Gateway, los cambios se guardan automáticamente.
